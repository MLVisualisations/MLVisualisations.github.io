Sorting Algorithms

This page is dedicated to sorting algorithms, which take arrays as inputs and output a new array which contains the same elements but with the entries in order from smallest to largest. That is, given an array $$ x = [x_1, x_2, x_3, \dots, x_n] $$ of unordered elements $x_i$, we apply a sorting algorithm Sort to obtain $$ \mbox{Sort}(x) = [x_{i_{1}}, x_{i_{2}}, \dots, x_{i_{n}}] $$ where $$ x_{i_{1}} \leq x_{i_{2}} \leq \dots \leq x_{i_{n}}. $$ Here $ i_{1}, i_{2}, \dots, i_{n} $ is simply a reordering of the original indices $ 1,2,\dots,n $. For example, if $$ x = [2, 5, 1, 8, 7], $$ then, $$ \mbox{Sort}(x) = [1, 2, 5, 7, 8]. $$
